============
Installation
============


Stable release
--------------

To install Water-Potability-Prediction, run this command in your terminal:

.. code-block:: console

    $ pip install water_potability_prediction_project

This is the preferred method to install Water-Potability-Prediction, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Water-Potability-Prediction can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Sudip-8345/water_potability_prediction_project

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Sudip-8345/water_potability_prediction_project/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Sudip-8345/water_potability_prediction_project
.. _tarball: https://github.com/Sudip-8345/water_potability_prediction_project/tarball/master
